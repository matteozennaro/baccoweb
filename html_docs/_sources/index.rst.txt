Bacco: 'Bias And Clustering Calculations Optimised'
==================================================

Here you will find the documentation for 'Bacco' -- a code to predict the dark
matter, halo, subhalo, and galaxy clustering from a fast cosmology scaling of 
N-body simulations.

Contents:

Basic Usage
--------------

This would scale a simulation from its original cosmology to another:

.. code-block:: python
  In [1]: import numpy as np

  In [2]: import bacco
